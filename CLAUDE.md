## Agent skills

### Issue tracker

Issues live in GitHub Issues (repo: Praful-Joshi/Renderit, via the `gh` CLI). External PRs are not treated as a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Default label vocabulary (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context: `CONTEXT-MAP.md` at the repo root points to `src/CONTEXT.md` (native C++/OpenGL renderer) and `web/CONTEXT.md` (Three.js web viewer). Root `docs/adr/` holds system-wide decisions. See `docs/agents/domain.md`.
